# 项目简介 java996

本项目是基于SpringBoot的物业管理系统的设计与实现，旨在为物业管理部门提供一个功能全面、易于操作的管理平台。

## 技术栈

- **开发工具**：IDEA
- **数据库**：MySQL 5或8
- **服务器**：Tomcat（任意版本）
- **后端框架**：Spring Boot
- **前端框架**：Vue、ElementUI、Layui

## 功能模块

- **后台登录**：保障系统的安全性，提供用户身份认证
  - 用户登录
  - 用户登出
- **房产信息管理**：管理物业内的房产信息
  - 查看房产信息
  - 新增房产信息
  - 编辑房产信息
  - 删除房产信息
- **房产类型管理**：维护房产类型数据
  - 查看房产类型
  - 添加房产类型
  - 修改房产类型
  - 删除房产类型
- **收费信息管理**：记录和计算各项收费
  - 收费项目管理
  - 收费金额设置
- **收费类型管理**：管理收费类型，便于分类查询
  - 查看收费类型
  - 添加收费类型
  - 编辑收费类型
  - 删除收费类型
- **用户管理**：对系统用户进行管理
  - 用户信息管理
  - 用户权限分配
- **租住信息管理**：跟踪房产的租住情况
  - 租住信息登记
  - 租住信息查询
- **车位管理**：维护车位使用信息
  - 车位信息管理
  - 车位分配

## 系统角色

- 系统管理员
- 物业工作人员
- 房产拥有者
- 租户

## 运行环境

- **操作系统**：支持Windows、Linux和MacOS
- **开发环境**：Java 1.8+
- **Web服务器**：Tomcat 7+

## 目录结构

```plaintext
项目根目录/
│
├── backend/              # 后端代码目录
├── frontend/             # 前端代码目录
└── sql/                  # 数据库脚本目录
```

## 部署步骤

1. 安装并配置MySQL数据库
2. 部署Tomcat服务器
3. 导入数据库脚本
4. 部署后端服务
5. 部署前端页面

- **高效率**：基于Spring Boot和Vue的前后端分离架构，提高系统响应速度和开发效率。
- **易用性**：简洁明了的用户界面，降低了用户的学习成本。
- **可扩展性**：合理的设计和模块化，方便后续功能的扩展和维护。
- **安全性**：系统的登录和权限控制，确保了数据的访问安全。
